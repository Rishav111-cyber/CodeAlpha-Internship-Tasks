# FAQ Chatbot

This project was developed as **Task 2** of the **CodeAlpha Artificial Intelligence Internship**.

The goal of this project is to build a simple FAQ chatbot that can answer predefined questions entered by the user.


## About the Project

The FAQ Chatbot allows users to:
- Ask common questions through a text input
- Get instant responses based on predefined FAQ data
- Interact with a clean and simple user interface

This project demonstrates how basic chatbot logic can be implemented without using complex machine learning models.


## Technologies Used

- Python
- Streamlit


## Project Structure

- `app.py` – Main Streamlit application file  
- `faq_data.py` – Contains predefined questions and answers  
- `requirements.txt` – Lists required dependencies  


## How to Run the Project

1. Install dependencies:
   ```bash
   pip install -r requirements.txt

2. Run the application:

python -m streamlit run app.py

3. Open the browser and go to:

http://localhost:8501

Output

The chatbot displays an input box where users can type questions such as:
What is this chatbot?
What is AI?
The chatbot responds instantly with the correct predefined answer.
